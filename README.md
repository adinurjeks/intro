# 🔐 Password Checker

Простая программа на Python для проверки сложности пароля.

## 📌 Описание

Программа проверяет, соответствует ли пароль базовым требованиям безопасности:

* минимум **2 цифры**
* минимум **7 букв**

Если пароль не подходит, программа сообщает об ошибке и объясняет причину.

---

## ⚙️ Функции программы

* Проверка количества букв и цифр
* Удобное меню
* Повторный ввод (программа работает в цикле)
* Обработка неправильного выбора

---

## ▶️ Запуск программы

1. Убедитесь, что установлен Python
2. Скачайте файл `main.py`
3. Запустите в терминале:

```bash
python main.py
```

## 🧠 Как работает программа

* Используется функция для проверки пароля
* Цикл `while` позволяет программе работать постоянно
* Меню помогает выбрать действие
* Используются условия `if/elif`
